# 老师爬楼梯问题
1. 每次可以爬1级或2级台阶
2. 输入楼梯的级数，求不同的走法数

分析：
    1： 1
    2： 1+1，2
    3： 1+1+1， 2+1， 1+2，  

f(1) = 1;
f(2) = 2;
f(3) = f(3-1) + f(3-2) = 3
...
f(n) = f(n-1) + f(n-2)

-------------------------------------------

- 大型项目注释写法
- 递归算法，可以优化
    n阶楼梯走法， fn = f（n-1） + f（n-2）
    递归的核心除了把大问题划分为N个相似的小问题，递过去，再归，解决
    一定要有退出条件
    最重要的是写递归公式

- 优化内存分配
